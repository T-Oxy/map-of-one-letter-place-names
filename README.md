# map-for-shiritori

## 概要
【しりとりで無双するための地図】
- データファイル
    - one-kana-place:  仮名表記の1文字の地名一覧
    - one-alpha-place:  アルファベット表記の1文字の地名一覧
    - nn-place:  「ん」から始まる地名一覧
    - di-place:  「ぢ」から始まる地名一覧

- 属性など
    - name_ja:  地名(漢字を含めた日本語表記)
    - name_en:  地名(英語表記)
    - name_kana:  地名(仮名表記)
    - place_name:  住所
    - geometry/coordinates:  緯度経度情報
